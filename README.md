# FitRank AI

FitRank AI is a hybrid candidate ranking system built for the India Runs Data & AI Challenge.

The system ranks candidates for a given job description using a combination of:
- semantic job-candidate matching
- skill and experience scoring
- career history analysis
- recruiter behavior signals
- risk and honeypot penalties

The goal is to produce a trustworthy top-100 candidate shortlist, not just keyword matches.
